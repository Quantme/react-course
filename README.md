## Projects

1. **Basic React App**  
   Introduction to React fundamentals  
   [`/basic`](/basic)

2. **Forms & API**
   Form handling and API integration
   [`/forms-api`](/forms-api)

3. **AI Chatbot**  
   API integration and advanced hooks  
   [`/chatbot`](/chatbot)
